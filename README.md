# EmeraldsBool

A simplistic boolean data type

# Installation

`em build lib`

## Usage

```c
#include "EmeraldsBool.h"

int main(void) {
  bool value = true;
  bool fvalue = false;
  if(value) {
    if(!fvalue) {
      printf("%s\n", "42");
    }
  }
}
```
